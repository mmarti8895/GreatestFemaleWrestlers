# Sources and Attribution

## Copyright and fair-use boundary

This project does **not** reproduce complete Pro Wrestling Illustrated (PWI) lists.
Checked-in PWI data is intentionally partial (for example, top-10 slices) and attribution-heavy.

## PWI references

- Pro Wrestling Illustrated official site: https://pwi-online.com/
- Yearly release materials for:
  - PWI Women's 150 (2022)
  - PWI Women's 250 (2023)
  - PWI Women's 250 (2024)

## All-time lists

All-time lists by promotion are curated in-repo using the scoring rubric documented in `methodology.html`.

## Extending data locally

Maintainers can extend `/data/pwi/` locally with additional yearly files and update `/data/pwi/index.json`.
To keep this repository fair-use-friendly, avoid committing full copyrighted lists.
